# Zeros

## Description
Ivancho's teacher gave him homework to write down numbers from 1 to $N$ in his notebook, but to make it more interesting, she gave him an additional task to count the zeros. Help Ivancho by writing a program that calculates how many times the digit zero has been used.

## Input
From the standard input, a value for $N$ is entered, a positive integer with at most 16 digits.

## Output
Output to the standard output how many times the digit zero has been used when writing all numbers from 1 to $N$, inclusive.

## Constraints
* $1 \le N \le 10^{16}$

## Examples

### Input
```
89
```

### Output
```
8
```

### Input
```
101
```

### Output
```
12
```

### Input
```
250
```

### Output
```
45
```

# Triangle

## Description
Write a program that inputs a number $N$ and outputs a triangle of $N$ rows. The outline of the triangle is made of asterisks, and the inside is made of dots. There is exactly one space between individual characters in the rows.

## Input
A number $N$.

## Output
A triangle of $N$ rows, with the outline of asterisks and inside of dots.

## Constraints
* $3 \le N \le 20$

## Example

### Input
```
5
```

### Output
```
*
* *
* . *
* . . *
* * * * *
```

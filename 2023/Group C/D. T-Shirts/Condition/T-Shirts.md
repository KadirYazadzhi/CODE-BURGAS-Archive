# T-Shirts

## Description
The organizers of a student programming contest wanted to give t-shirts to all participants.
However, to order the t-shirts for the competitors, they had to provide the manufacturer with information about the height in centimeters of the shortest and the tallest student participating in the competition.
Help the organizers of the student programming contest by writing a program that finds the height in centimeters of the shortest and the tallest student.

## Input
From the first line of the standard input, a single integer $n$ is entered, which corresponds to the number of students participating in the competition.
On the next $n$ lines from the standard input, integers are entered, corresponding to the height in centimeters of each of the students participating in the competition.

## Output
On a single line of the standard output, the program should output two numbers, corresponding to the height in centimeters of the shortest and the tallest student participating in the competition, separated by a single space.

## Example

### Input
```
5
160
140
180
170
150
```

### Output
```
140 180
```

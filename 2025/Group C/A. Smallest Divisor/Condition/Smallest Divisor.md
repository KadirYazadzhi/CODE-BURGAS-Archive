# Smallest Divisor

## Description
Write a computer program that accepts an integer $N$ and outputs the smallest natural divisor of $N$ other than one.

## Input
On a single line of the standard input, an integer $N$ is entered.

## Output
On a single line of the standard output, output the smallest divisor of $N$ other than one.

## Example

### Input
```
15
```

### Output
```
3
```

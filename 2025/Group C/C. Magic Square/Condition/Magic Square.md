# Magic Square

## Description
Given an $N \times N$ square matrix. Check if it is a magic square, meaning whether the sum of the numbers in each row, each column, and both main diagonals is the same.

## Input
On the first line, an integer $N$ is entered for the size of the square matrix.
On the next $N$ lines, the numbers of the matrix are entered, separated by spaces.

## Output
If the matrix is a magic square, output `YES`, otherwise `NO`.

## Example

### Input
```
3
8 1 6
3 5 7
4 9 2
```

### Output
```
YES
```

# SUMS

## Description
A given number of positive integers are provided. Calculate the sums of all possible consecutive pairs of digits for each given positive integer and output the minimum and maximum among the obtained sums.

## Input
The first line contains the number $N$ of positive integers.
The next $N$ lines each contain a positive integer.

## Output
For each positive integer, a new line is output to the standard output containing the minimum and maximum sums obtained, separated by a space.

## Example

### Input
```
2
12345
11
```

### Output
```
3 9
2 2
```

### Explanation
1st number: min (1+2) = 3; max (4+5) = 9
2nd number: min (1+1)=2; max (1+1) = 2

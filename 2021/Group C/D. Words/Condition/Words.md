# WORDS

## Description
Ivancho suggested to Mariyka a new game in which he invents new words, and she has to say which word has the fewest lowercase letters and which word has the most uppercase letters. Help Mariyka win Ivancho's game.

## Input
The first line contains the number $N$ of words.
The next $N$ lines contain the words themselves.

## Output
Output to the standard output: the word with the fewest lowercase letters and the word with the most uppercase letters, separated by a newline.

## Example

### Input 1
```
3
AAAaa
a
Aaaaa
```

### Output 1
```
a
AAAaa
```

### Input 2
```
1
Aa
```

### Output 2
```
Aa
Aa
```

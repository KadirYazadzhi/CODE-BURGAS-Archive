# Binary Numbers

## Description
Since the students from Hogwarts School are very good at programming and mathematics, they quickly solve all problems and get bored easily, so they decided to compete. They started calculating in binary numeral system and betting on the number of ones in the binary representation of a number. Help the students determine the winner of the competition.

## Input
On the first line, the number of students and the decimal number are entered, separated by a space.
On the second line, each student's guess for the number of ones in the binary representation of the number is entered.

## Output
Output to the standard output the number of the student with the closest result (if there is more than one, output them separated by a space on the same line, in the order they were entered).

## Example

### Input
```
3 6
3 2 1
```

### Output
```
2
```

# Pandemic

## Description
Since Teddy's school is closed and conducts online learning, children find it difficult and their performance has decreased. Fortunately, Teddy has a good teacher who decided to give a bonus on the test to those who have a birthday in the month the test is held â€“ to raise their grades by 1 unit, unless, of course, they have a 6. Help the teacher determine which children should have their grades raised.

## Input
The first line contains the birthday (day and month), separated by spaces.
The second line contains the test month and grade, separated by spaces.

## Output
Output to the standard output the grade the student should receive.

## Example

### Input 1
```
10 4
3 2
```

### Output 1
```
2
```

### Input 2
```
10 4
4 3
```

### Output 2
```
4
```

# Warm-Up

## Description
The weather in Burgas has been warming up over the last month, but there are also days when it gets colder. The daily temperatures for the last $N$ days are provided on the standard input. Determine the maximum number of consecutive days during which the temperatures are rising, i.e., each day the temperature is higher than the previous one.

## Input
On the first line of the standard input, the number $N$ of days is given.
The following $N$ lines contain numbers specifying the temperatures for $N$ consecutive days.

## Output
Output the maximum number of consecutive days with increasing temperatures to the standard output.

## Example

### Input
```
6
10 12 13 13 14 11
```

### Output
```
3
```

# Number Systems

## Description
Find the minimum possible base $M$ for which the equation $X + Y = Z$ holds true.

## Input
The first line of the standard input contains the number $N$ of test cases. 
The next $N$ lines contain the test cases. 
Each test case consists of a single line containing the equation: $X + Y = Z$.

## Output
For each test case, output to the standard output the minimum base $M$ for which the equation $X + Y = Z$ is solved.

## Constraints
* $1 \le N \le 100$
* $2 \le M \le 36$
* $1 \le \text{Length}(X) \le 10$
* $1 \le \text{Length}(Y) \le 10$
* $1 \le \text{Length}(Z) \le 10$

## Example

### Input

```
2
01+10=11
52+38=101
```

### Output

```
2
9
```

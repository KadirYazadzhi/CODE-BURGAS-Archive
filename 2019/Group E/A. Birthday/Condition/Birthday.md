# Birthday

## Description
Dani was very excited about her birthday, so from the beginning of the year, she counted the days until it. She infected her friends with her excitement, who also tried to calculate how many days were left for them, although they were not as good as her. Help the girls calculate how many days are left until their birthdays by creating a program that calculates how many days there are between 2 arbitrary dates within the same year.

## Input
The first line contains the start day and month, separated by spaces.
The second line contains the end day and month, separated by spaces.

## Output
Output the number equal to the number of days between the two dates to the standard output.

## Example

### Input
```
2 2
3 2
```

### Output
```
1
```

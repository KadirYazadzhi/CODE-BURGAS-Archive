# Codograd Square

## Description
In Codograd, the time has come to allocate space for a square. Since the administration of Codograd does not like half-measures, the mayor of Codograd specially orders you to write a program that calculates how many people the square can accommodate, if it is square-shaped and 4 people can stand on one square meter.

## Input
The first line of the standard input contains the number $T$ of test cases. Each test consists of 1 line.
The line contains 1 integer $N$ â€“ this number shows the side length of the Codograd square.

## Output
For each test, your program must output on a separate line the number of people the square can accommodate.

## Constraints
* $N$ will be in the range $[1; 10000]$.

## Example

### Input

```
3
50
500
1000
```

### Output

```
10000
1000000
4000000
```

### Explanation
Area = $50 \times 50 = 2500$ sq.m.
People = $2500 \times 4 = 10000$.

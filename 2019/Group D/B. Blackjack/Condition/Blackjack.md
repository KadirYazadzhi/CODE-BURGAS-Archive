# Blackjack

## Description
As you may know, in the popular game BLACKJACK, the goal is to collect cards whose sum is as close to 21 as possible. In the game we will play, the player is your program, which must read $N$ integers. From these numbers, you will have to choose three, so that the sum of these numbers is as close as possible to, but not exceeding, a given number $M$.

## Input
The first line of the standard input contains the number $N$ of numbers.
The next line contains the numbers themselves.
The third line contains $M$.

## Output
Output the obtained sum.

## Constraints
* $3 \le N \le 100$
* $10 \le M \le 300000$

## Example

### Input
```
5
3 12 8 1 7
19
```

### Output
```
18
```

### Explanation
$3 + 8 + 7 = 18$

# Maximum Sum

## Description
Given an array of integers. Find the largest sum you can get from a subarray of this array without using two consecutive elements.

## Input
The first line of the standard input contains the number of elements in the array.
The second line of the standard input contains the elements of the array separated by a space.

## Output
Output the obtained maximum sum on a single line of the standard output.

## Example

### Input
```
5
1 2 3 4 5
```

### Output
```
9
```

### Explanation
The largest sum is obtained from the subarray: 1 3 5 ($1+3+5=9$).
